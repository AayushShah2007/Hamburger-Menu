# HamburgerMenu

A simple, responsive hamburger navigation menu built with HTML and CSS for a restaurant website called **Mezzatera**.

## Features

- Hamburger icon triggers a sliding sidebar on hover
- Sidebar contains navigation links: Home, Menu, Combo Offers, Reserve a Table, Reviews, About Us
- Login / Sign Up links in the top-right corner
- Smooth CSS transition animation for the sidebar slide-in effect
- Modern styling with Font Awesome icons

## How to Use

Open `HamburgerMenu1.html` in any modern web browser. Hover over the hamburger icon (☰) to reveal the sidebar menu.

## Structure

```
HamburgerMenu/
├── HamburgerMenu1.html   # Main HTML file with inline CSS
└── README.md
```

All styles are embedded within the HTML file — no external CSS dependencies. The page uses Font Awesome via CDN for icons.
